# Dynamic Wumpus Logic Agent

## Overview
This project implements a Knowledge-Based Agent in Wumpus World using Propositional Logic and Resolution.

## Features
- Dynamic Grid
- Random Wumpus & Pits
- Percepts (Breeze, Stench)
- Knowledge Base
- Inference Steps
- Web Interface (Flask)
